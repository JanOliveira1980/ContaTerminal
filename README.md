## Simulando uma Conta Bancaria

Pequena aplicação simulando entrada de dados,processamento e saida em uma conta bancaria, 
consolidando conceitos basicos da linguagem JAVA.

## Como Usar

Entre com os dados requisitados no termnal, pressione ENTER e tenha a exibição dos seus dados bancarios em tela
